IEC60870-5-101/104 Master IO table
==================================

This section describes how to create an IO object table for IEC60870-5-101 and IEC60870-5-104 controlling station (Master) communication protocol instances.
Group object nodes :ref:`xmlgroup-IEC10xmaDI`; :ref:`xmlgroup-IEC10xmaAI`; :ref:`xmlgroup-IEC10xmaDO`; :ref:`xmlgroup-IEC10xmaAO` are
included in XML configuration files of the IEC60870-5-101 and IEC60870-5-104 controlling station (Master) communication protocol instances as described in previous 
sections :ref:`docref-IEC101ma` and :ref:`docref-IEC104ma`.
