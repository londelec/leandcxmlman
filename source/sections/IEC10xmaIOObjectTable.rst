IEC60870-5-101/104 Master IO object table
=========================================

This section describes how to configure IO object table for IEC 60870-5-101 and IEC 60870-5-104 controlling 
station (Master) communication protocol instance. Group object nodes :ref:`DITable<ref-IEC10xmaDITable>`; :ref:`AITable<ref-IEC10xmaAITable>`; :ref:`DOTable<ref-IEC10xmaDOTable>`; :ref:`AOTable<ref-IEC10xmaAOTable>`
are included in XML configuration files of the IEC60870-5-101 and IEC60870-5-104 controlling station (Master) 
communication protocol instances as described in previous 
sections :ref:`docref-IEC101maConfig` and :ref:`docref-IEC104maConfig`.
